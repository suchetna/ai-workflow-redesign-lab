# AI Workflow Redesign Lab

Workflow redesign patterns for integrating AI into research, content, operations, and decision-making.

## Why this exists

Most organisations do not get value from AI just by adding a tool to an existing process. The real work is redesigning workflows so AI supports better decisions, faster execution, and clearer handoffs.

This repository is a working lab for developing practical workflow redesign patterns for teams adopting AI.

## What this repository will include

- Before-and-after workflow maps
- AI-assisted process redesign templates
- Role and responsibility shift frameworks
- Human-in-the-loop review checkpoints
- Workflow examples for research, content, operations, and strategy

## Current status

This repository is being built as a public working library. The first files will focus on workflow mapping, review checkpoints, and team redesign patterns.

## Working principle

Do not layer AI on top of broken workflows. Redesign the workflow first.
